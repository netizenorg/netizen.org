Whois netnet?

say howwwdy!

Handcrafted with <3 by the folx at netizen.org (that’s us!) netnet is a friendly AI + code editor at the center of netnet.studio, a hypermedia online space for fully realizing the Web’s creative potential. At its core, netnet.studio is a code playground: a didactic, real-time, and augmented web development environment. There are a few different points of entry into this space for BrowserFest:

interactive tutorials

Get things flowing by going through the interactive tutorials which should get you oriented with netnet as well as introduce you to our particular approach to creative coding. These lessons will not only teach you how to craft a website through its core languages (HTML, CSS, and JavaScript) but also what it means to use these languages in unconventional ways to produce web demos.

interactive examples

Explore our starter packs of code snippets and examples you might find useful in building out your demos. You can edit and remix these to get a feel for how the code works. Double-click any piece of code if you’d like netnet to explain what it’s for.

creative playground

Feel free to jump straight into making your demos if you’ve already got some ideas, but we suggest watching this video first to get oriented with the studio. netnet is modeled after other popular code editors, so if you’ve used something like Atom, VSCode or SublimeText it should feel familiar. But being a web-based, pedagogically oriented real-time playground, there are a few differences you might want to learn about before you dive in.
